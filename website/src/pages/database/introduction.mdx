# Introduction

Your GraphQL API can connect to any kind of data source, such as a remote REST (or even GraphQL) API or database.

## Schema

Each of the [GraphQL schemas](/get-started#schema) will used as chapters for each of the database implementations.

Furthermore, every database implementation that follows has a resulting `schema.ts` file containing an async `buildSchema('<schema>')` function that is used to communicate to the database over GraphQL.

<Callout>
  Notice the usage of the argument passed to the `buildSchema` (`'basic'`,
  `'subscriptions'`, etc.) function to decide which schema to build.
</Callout>

### [Basic](/get-started#basic)

```ts
import { GraphQLSchema } from 'graphql';
import { buildSchema } from '@database/<database>/schema';

const BasicSchema: GraphQLSchema = await buildSchema('<schema>');
```

All databases in the [database](/database) module use the port **50000** ([http://localhost:50000](http://localhost:50000)).
