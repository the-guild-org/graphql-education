# Introduction

Your GraphQL API can connect to any kind of data source, such as a remote REST (or even GraphQL) API or database.

## Schema

Each of the [GraphQL schemas](/get-started#schema) will be used as chapters for each of the database implementations.

Furthermore, every database implementation that follows has a resulting `<schema>.ts` file containing an asynchronous `buildSchema()` function that is used to generate the GraphQL executable schema that communicates to the database over GraphQL.

### [Basic](/get-started#basic)

```ts
import { GraphQLSchema } from 'graphql';
import { buildSchema } from '@database/<database>/schema/basic';

const BasicSchema: GraphQLSchema = await buildSchema();
```

### [Subscriptions](/get-started#subscriptions)


All databases in the [database](/database) module use the port **50000** ([http://localhost:50000](http://localhost:50000)).
